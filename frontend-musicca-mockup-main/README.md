# Frontend Musicca
> This is a frontend created for the Web Programming TB 3 classes at École des Mines Saint-Étienne

## Requirements
- Python3
- SASS (for scss)

## How to install SASS
```bash
npm install -g sass
```

## How to run

```bash
./integration/compile_css.sh
./run_server.py
```

Now you just need to go to https://localhost:8000
